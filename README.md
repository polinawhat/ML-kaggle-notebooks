# HSE University — Machine Learning & Applied Data Analysis Portfolio

Welcome! This repository contains a collection of end-to-end machine learning and data analysis projects developed during my studies at the Faculty of Computer Science, HSE University (National Research University Higher School of Economics). 

The projects demonstrate practical application of ML algorithms across diverse data domains: tabular data, multi-output regression, high-dimensional spectral analysis, computer vision, and bioinformatics.

## 📁 Repository Structure & Projects

* **`01-vehicle-price-forecasting/` — Vehicle Price & Fuel Efficiency Forecasting**
  * *Task:* Multi-output regression pipeline predicting vehicle cost and MPG based on structural attributes.
  * *Keywords:* Feature engineering, multicollinearity, grid search CV, regression.

* **`02-stellar-classification/` — Stellar Entity Classification via Spectral Analysis**
  * *Task:* Classifying astronomical objects using high-dimensional noisy spectral features.
  * *Keywords:* Imbalanced classes, F1-score optimization, feature selection.

* **`03-digit-recognizer/` — Handwritten Digit Recognizer**
  * *Task:* Binary image classification (distinguishing digits 3 and 8) from raw pixel data.
  * *Keywords:* Pixel normalization, dimensionality reduction, confusion matrix.

* **`04-phoneme-signal-classification/` — Phoneme Audio Signal Classification**
  * *Task:* Identifying specific phoneme sounds ('g') using digital signal frequency-domain representations.
  * *Keywords:* Acoustic patterns, signal processing, tabular matrices.

* **`05-genomic-sequence-classification/` — Genomic Sequence Classification**
  * *Task:* Classifying bacterial strains directly from raw biological DNA sequences.
  * *Keywords:* String tokenization, k-mer counting, text-mining pipeline.
